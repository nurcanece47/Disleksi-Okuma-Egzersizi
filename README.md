# Disleksi-Okuma-Egzersizi

Kodları Şu şekilde tasarlandı hangi lbok kodların hangi sayfaya ait olduğunu üste screen 1 gbi yazılı olacaktır ordan bakarak karışıklık olmadan yapabilirsiniz. Bu size kolaylık sağlayacaktır. Önce tasarımı sonra hemen altında kodları olacak şekilde tasarlandı bu uygulama Mit app invertor uygulamasında tasarlandı ordan tasarlamanızı tavsiye ederim.
